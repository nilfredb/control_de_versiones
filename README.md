## 🌿 Metodología Git Flow aplicada

El proyecto implementa correctamente el flujo de trabajo Git Flow con las siguientes ramas:

- `main` → Producción
- `dev` → Desarrollo
- `qa` → Pruebas

### Ramas utilizadas:

- `feature/create-user`
- `feature/list-users`
- `feature/edit-user`
- `feature/delete-user`
- `hotfix/fix-form-validation`

## ⚙️ Instalación y ejecución

```bash
# Clonar repositorio
git clone https://github.com/TU-USUARIO/control_de_versiones.git

# Entrar al proyecto
cd control_de_versiones

# Crear entorno virtual
python -m venv venv

# Activar entorno (Windows)
venv\Scripts\activate

# Instalar dependencias
pip install -r requirements.txt

# Ejecutar aplicación
python app.py
```
